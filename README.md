# 智慧社区物业管理系统

## 项目简介

智慧社区物业管理系统是一个基于前后端分离架构的现代化物业管理平台，旨在为社区管理员和业主提供便捷的管理和服务体验。系统采用Vue3 + SpringBoot3 + MySQL8技术栈，实现了业主管理、楼栋房屋管理、报修管理、费用管理、公告管理、投诉建议、设施管理、停车管理等核心功能。

## 技术栈

### 前端
- Vue3 + Vite + Element Plus + Pinia + Vue Router + Axios + ECharts

### 后端
- SpringBoot3 + MyBatis-Plus + MySQL8 + Spring Security + JWT + Knife4j

### 数据库
- MySQL 8.0

## 系统架构

- 前后端分离架构
- RESTful API接口设计
- RBAC权限管理
- 响应式布局

## 功能模块

### 管理员端
1. 登录/注册/退出（JWT认证，支持密码重置）
2. 首页数据看板（ECharts可视化）：住户总数、待处理报修、本月缴费、欠费统计、设施完好率
3. 业主管理：业主信息增删改查、楼栋房号关联、业主状态管理、信息导出Excel
4. 楼栋房屋管理：楼栋、单元、房屋、户型、面积、房屋状态（空闲/已入住）管理
5. 报修管理：报修列表查看、报修处理、状态更新（待处理/处理中/已完成）、业主评价查看
6. 费用管理：物业费/水电费/停车费生成、账单管理、缴费记录查询、欠费提醒、缴费统计
7. 公告管理：公告发布、编辑、删除、置顶、草稿保存
8. 投诉建议：查看业主提交的投诉建议、回复处理、处理记录留存
9. 设施管理：公共设施信息维护、设施维护记录、设施故障上报处理
10. 停车管理：车位信息、业主车辆登记、出入记录查询
11. 权限管理：角色分配、菜单管理、用户权限控制（适配管理员单一管理角色）
12. 个人中心：个人信息修改、密码修改、操作日志查看

### 业主端
1. 注册、登录、实名认证（绑定房屋信息）
2. 首页：最新公告展示、报修快速入口、待缴费提醒、个人房屋信息
3. 我的报修：提交报修申请、查看报修进度、取消报修、报修完成后评价
4. 我的缴费：待缴费账单查看、在线支付（模拟支付流程）、缴费历史记录查询
5. 投诉建议：提交投诉/建议、查看回复内容、跟踪处理进度
6. 房屋信息：查看个人房屋详情、家庭成员信息管理、房屋绑定修改
7. 个人中心：个人信息编辑、密码修改、消息通知（报修/缴费/公告提醒）

## 快速开始

### 环境要求
- JDK 17+
- MySQL 8.0+
- Node.js 16+
- npm 7+

### 后端启动
1. 导入数据库脚本：`db/property_management.sql`
2. 配置数据库连接：修改 `backend/src/main/resources/application.yml` 中的数据库连接信息
3. 启动后端服务：运行 `backend/src/main/java/com/property/Application.java`
4. 访问接口文档：`http://localhost:8080/api/doc.html`

### 前端启动
1. 安装依赖：`cd frontend && npm install`
2. 启动开发服务器：`npm run dev`
3. 访问前端页面：`http://localhost:3000`

### 测试账号
- 管理员账号：admin / 123456
- 业主账号：owner / 123456

## 注意事项

1. 系统使用JWT进行身份认证，请确保在请求头中携带有效的token
2. 前端开发服务器配置了代理，将API请求代理到后端服务
3. 数据库脚本包含了初始数据，方便测试使用
4. 系统支持响应式布局，适配不同屏幕尺寸
